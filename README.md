##  Setup

 - ```cp .env.example .env```

- ```composer install```

- ```npm install```

- ```npm run dev ```

- ```php artisan key:generate ```

- ```php artisan storage:link ```

- ```php artisan sail:install ```

- ```./vender/bin/sail up ```

- ```npm run dev ```

- ```./vender/bin/sail artisan migrate --seed ```
